# Dapr 运行在openEuler
Dapr 是一个可移植的、事件驱动的运行时，它使任何开发人员能够轻松构建出弹性的、无状态和有状态的应用程序，并可运行在云平台或边缘计算中。

**More details info, Please access [link](https://cn.dapr.io/).**
